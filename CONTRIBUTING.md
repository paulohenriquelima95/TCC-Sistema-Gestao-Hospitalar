# Guia de Contribuição

Obrigado pelo seu interesse em contribuir para o projeto **Sistema de Gestão Hospitalar e de Serviços de Saúde (SGHSS)**. Este projeto é o Trabalho de Conclusão de Curso (TCC) de Paulo Henrique da Luz Lima, com foco em back-end utilizando Python e Flask.

Embora o projeto tenha sido concluído para fins acadêmicos, ele está aberto a melhorias e evoluções.

## Como Contribuir

### 1. Sugestão de Recursos e Ideias

Se você tem uma ideia para uma nova funcionalidade, melhoria de código ou correção de bug, por favor, abra uma **Issue** no repositório.

*   **Bugs:** Se encontrar um erro, use o template de "Bug Report" e inclua o máximo de detalhes possível (passos para reproduzir, comportamento esperado e comportamento atual).
*   **Sugestões:** Para novas funcionalidades ou melhorias, use o template de "Feature Request" e descreva o objetivo, a justificativa e o impacto da mudança.

### 2. Contribuição de Código

Se você deseja contribuir com código, siga os passos abaixo:

1.  **Faça um Fork** do repositório para sua conta pessoal.
2.  **Clone** o seu fork para sua máquina local.
3.  Crie um **Branch** para sua contribuição (ex: `feature/adicionar-relatorio-x` ou `fix/correcao-autenticacao`).
4.  Faça suas alterações no código.
5.  Certifique-se de que o código segue as diretrizes de estilo (PEP 8 para Python) e que todos os testes existentes (se houver) continuam passando.
6.  **Commit** suas alterações com mensagens claras e descritivas (ex: `feat: implementa endpoint de relatórios de agendamento`).
7.  **Push** o branch para o seu fork no GitHub.
8.  Abra um **Pull Request (PR)** do seu branch para o branch `main` do repositório original.

### 3. Diretrizes de Pull Request

*   O título do PR deve ser conciso e refletir a mudança.
*   A descrição do PR deve fazer referência à Issue que ele resolve (ex: `Closes #123`).
*   Mantenha os PRs focados em uma única funcionalidade ou correção.

## Padrões de Código

Como o projeto utiliza **Python** e **Flask**, esperamos que o código siga as convenções da **PEP 8**.

Agradecemos imensamente sua colaboração!
